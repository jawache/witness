# Witness Test Vault

This is a test vault for the Witness plugin development.

## Structure

- `chaos/` - Unprocessed, incoming information
  - `external/` - From outside world
  - `inbox/` - Quick notes and ideas
- `order/` - Structured, processed information  
  - `knowledge/` - Databases of things
  - `heartbeat/` - Daily, weekly, quarterly rhythms
  - `projects/` - Active work
  - `synthesis/` - Your voice and output
